關於COSCUP 2018 X openSUSE.Asia X GNOME.Asia

開源人年會 COSCUP 已經邁入第十三年了，今年的活動有些特別，除了延續去年首次嘗試的社群議程外，今年更與 GNOME.Asia 與 openSUSE.Asia 一起合辦，今年的時間地點則於 8/11-12 在台灣科技大學國際大樓舉辦。過去，COSCUP 由各地社群共同舉辦，在眾多同好的參與下，逐漸成長茁壯，今年除了有在地社群成為議程的協力夥伴外，更加入了國際社群們，希望能打造更多元的主題和議程。 無論你是寫 Open Source 軟體的開發者、熱衷 Open Source 軟體的推廣者、熟悉 Open Source 軟體的使用者或是想了解 Open Source 軟體的新手，都歡迎你來參加一年一度全台最大的 COSCUP x GNOME.Asia x openSUSE.Asia 2018！

COSCUP 2018 議程介紹

COSCUP 2017 以「擁抱社群，回歸創始，莫忘初衷，擁抱開源」為主題，COSCUP 與 22 個開源社群一同協調規劃今年的 8 軌議程，超過 140 段議程輪番登場，網羅目前台灣熱門議題，吸引來自世界各地講者，帶來包含開放農業、物聯網技術與開發、資料庫的應用與研究、全端網頁技術的優化、Android App 開發、分散式與海量計算電腦區塊鏈、作業系統開發與選擇、電腦字型發展、社群經營、開放 API 與資訊安全。除了上述這些議題，COSCUP 也有準備輕鬆愉快的開源歷程分享，讓第一次參與 COSCUP 的與會者們瞭解其他人的開源之路。
