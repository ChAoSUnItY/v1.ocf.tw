網路干擾，包含網站存取的阻擋、插入惡意程式碼等等，是現今各國政府用於控制資訊的手段之一，最知名的網路干擾計劃，莫過於中國的防火長城，封鎖中國境內不得存取維基百科、 Google, BBC 等網路上重要的資訊和消息來源，並偵測及封鎖各種翻牆軟體。除了防火長城之外，各國政府或多或少均有使用各種技術封鎖不同網站，理由可能包含著作權侵害、散佈假消息、散佈不利於政府的資訊等等。網路封鎖阻擋了資訊的自由流通，經常也會影響到人民的言論自由、知情權、祕密通訊自由等基本權利。瞭解網路封鎖的狀況即可瞭解一部分政府在網路上侵犯公民權利的狀況。

公民要如何瞭解網路封鎖的現狀和趨勢呢？要瞭解政府阻擋了哪一些網站，是否有一套自動化的檢查程式？政府封鎖與開放特定網站，也會因為國家、地區、電信商、時間有所不同，是否有方法彙整不同條件下檢查程式的結果，讓我們看出趨勢？

解答以上這些問題，即是「網路干擾開放測量計劃」（Open Observatory of Network Interference, OONI）成立的初衷。

什麼是 OONI ?

OONI 是全世界最大的公開測量、研究網路封鎖的計劃，也是 Tor Project （洋蔥路由器）旗下的自由軟體專案。OONI 計劃涵蓋一套跨手機/電腦平臺的網路封鎖偵測程式「OONI Probe」，以及後端的資料庫和分析工具，可以將蒐集到的偵測資料，彙整進他們旗下的全球網路審查情形資料庫，內容包含：

各國政府封鎖網站的情況
各國政府禁用即時通訊軟體的情況（例如：Telegram）
各國政府禁用反封鎖工具（翻牆工具）的情況（例如：Tor）
網路速度與效能
網路中可能存在的審查/監視系統（中間盒）

自 2012 年起，OONI 已經從全球 200 多個國家，收集了數百萬筆網路測量資料，所有資料都發佈在 OONI Explorer 和 OONI API 上。

開放文化基金會為 OONI 的正式夥伴，除了在辦公室設置 OONI 偵測器之外，也將共同改進 OONI 網路干擾的測試清單、協助分析 OONI 收集的網路干擾測量資料。
